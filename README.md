Projekt Idee:
Eine Webanwendung, die wie ein Freundebuch funktioniert und Daten wie Name, Alter, Geburtstag, 
Lieblingsfarbe, Lieblingsessen, Hobbys und Traumjob speichert.

geplante/Idee für Use Cases:
- Benutzerregistrierung und -anmeldung
- Profil erstellen und bearbeiten 
- Freundebuch-Einträge filtern nach bestimmten Kategorien und durchsuchen
- kommende Gebursttage im Kalender anzeigen lassen
- Favoriten markieren

Ideen für Funktionen für Zusatzpunkte:
- Input Validierung (Geburtstag und Alter passen zsm ect.)
- Multi User
- externe Api (z.B Kalender Api für Geburtstage)
